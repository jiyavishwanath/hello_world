# 🌍 Hello World Program

## 📌 Description

This is a simple C program that prints **"Hello, World!"** on the screen.

## 💻 Code

```c
#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}
```
## OUTPUT
HELLO WORLD!

## ▶️ How to Run

1. Save file as `hello.c`
2. Compile:
   gcc hello.c -o hello
3. Run:
   ./hello
